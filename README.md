# DOUBLE RINGS - Визуализатор интерьеров

Интерактивный визуализатор бамбуковых панелей и отделочных материалов.

## 🚀 Запуск

Просто откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

Затем перейдите по адресу: `http://localhost:8000`

## 📁 Структура

```
interior-visualizer/
├── index.html              # Основной файл
├── images/
│   └── sofa.png            # Изображение дивана (PNG с прозрачностью)
├── bamboo_config.json      # Конфигурация материалов
├── generate_bamboo_config.py  # Скрипт генерации конфига из CSV
└── generate_sofa.py        # Скрипт генерации изображения дивана
```

## 🎨 Особенности

- **83 материала** из 6 категорий:
  - Cерия ткань (13 текстур)
  - Cерия дерево (14 текстур)
  - Cерия метал (14 текстур)
  - Cерия мрамор (14 текстур)
  - Cерия блеск кристала (14 текстур)
  - Cерия кожа (14 текстур)

- **Адаптивный дизайн** — работает на всех устройствах
- **Сохранение выбора** — состояние сохраняется в localStorage
- **Угловой диван** — PNG изображение с прозрачным фоном
- **Ковровая текстура** — имитация пола на широких экранах

## 🛠️ Технологии

- HTML5 + CSS3 + Vanilla JavaScript
- Без внешних зависимостей
- GitHub Pages для хостинга
